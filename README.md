# WTF Langchain

最近在学习Langchain框架，顺手写一个开源教程。本教程默认以下前提：
- 使用Python版本的[Langchain](https://github.com/hwchase17/langchain)
- LLM使用OpenAI的模型
- Langchain目前还处于快速发展阶段，版本迭代频繁，为避免示例代码失效，本教程使用最新版本 **0.0.235**
